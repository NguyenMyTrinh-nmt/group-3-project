# Giới thiệu dự án
Dự án này được thực hiện nhằm mô phỏng một hệ thống **quản lý người dùng (User Management System)** với đầy đủ chức năng **CRUD (Create – Read – Update – Delete)**.  
Ứng dụng được xây dựng dựa trên kiến trúc **Frontend – Backend – Database** nhằm giúp nhóm làm quen với cách thiết kế một ứng dụng web hoàn chỉnh từ giao diện, xử lý dữ liệu đến API phía máy chủ.  

Mục tiêu chính:
- Hiểu và áp dụng quy trình phát triển phần mềm theo nhóm sử dụng Git và GitHub.  
- Thực hành xây dựng RESTful API bằng **Node.js + Express**.  
- Xây dựng giao diện trực quan bằng **ReactJS**.  
- Thiết kế cơ sở dữ liệu và mô hình hóa dữ liệu người dùng.  

---

# Công nghệ và công cụ sử dụng

| Thành phần | Công nghệ / Công cụ | Mô tả |
|-------------|---------------------|-------|
| **Frontend** | ReactJS, Axios, TailwindCSS | Giao diện web, gọi API, xử lý sự kiện người dùng |
| **Backend** | Node.js, Express | Tạo REST API, xử lý logic CRUD |
| **Database** | MongoDB  | Lưu trữ dữ liệu người dùng |
| **Công cụ hỗ trợ** | Git, GitHub, Postman, Visual Studio Code | Quản lý mã nguồn, test API và phát triển code |
